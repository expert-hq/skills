# Expert Skills

Agent skills for escalating to human experts when AI gets stuck.

## Quick Start

```bash
npx skills add mblode/expert-skills -g --all -y
```

Supports Claude Code, Codex, OpenCode, and Cursor.

## Skills

| Skill | Phase | What it does |
|-------|-------|--------------|
| `expert` | Build | Book a paid call ($50 AUD) with a human expert via Cal.com when the AI is stuck |
